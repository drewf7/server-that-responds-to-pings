# A Server That Responds To Ping
That's it. Listens on every path. Responds "Pong" to any request.

Logs requesting path and ip.

## Installing
Requires NodeJS (Tested on 10.15.1)

To setup simply
```
npm install
```

## Testing
Run
```
npm test
```
